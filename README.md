# ♟️ Python Chess Game

A simple **Chess Game built using Python** where two players can play chess on the same computer. The game follows standard chess rules and includes piece movement, capturing, and turn-based gameplay.

---

## 📌 Features

* ♜ Full chess board with all pieces
* ♞ Valid movement rules for each piece
* ♝ Turn-based gameplay (White vs Black)
* ♛ Piece capturing
* ♚ Basic check and checkmate detection
* ♟ Simple user interface

---

## 🛠️ Technologies Used

* **Python 3**
* **Pygame**

---

## 📂 Project Structure

```
chess-game/

├── main.py        # Runs the game
├── board.py       # Board logic and rendering
├── pieces.py      # Chess piece movement rules
├── utils.py       # Helper functions
└── assets/        # Images for chess pieces
```

---

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/yourusername/python-chess-game.git
```

2. Navigate to the project folder

```
cd python-chess-game
```

3. Install dependencies

```
pip install pygame
```

---

## ▶️ How to Run

Run the following command:

```
python main.py
```

The chess game window will open and you can start playing.

---

## 🎮 How to Play

* Click on a piece to select it
* Click on a valid square to move the piece
* Players take turns moving pieces
* Capture opponent pieces by moving onto their square
* The game ends when **checkmate** occurs
